# ⚡ Three-Phase Vienna Rectifier Project

## 🚀 Overview

This project focuses on the **design and simulation of a three-phase three-level boost-type Vienna rectifier** aimed at improving input power quality and reducing harmonic distortion.

The Vienna rectifier is widely used in **telecom power supplies, EV chargers, and industrial applications** due to its high efficiency and near-unity power factor.

---

## 🎯 Objectives

* Improve input power factor
* Reduce Total Harmonic Distortion (THD)
* Achieve efficient AC–DC conversion
* Analyse system performance using simulation

## ⚙️ Working Principle

The Vienna rectifier is a **three-level PWM rectifier** that converts AC to DC efficiently while maintaining high power quality.

* Uses fewer active switches compared to conventional rectifiers
* Operates with near unity power factor
* Reduces switching and conduction losses
* Minimizes harmonic distortion in input current
