# Vienna-rectifier-project
# ⚡ Three-Phase Vienna Rectifier

## 🚀 Overview
This project focuses on the design and simulation of a **three-phase three-level boost-type Vienna rectifier** to improve input power quality and reduce harmonic distortion.

It is widely used in **telecom power supplies, EV chargers, and industrial applications**.

---

## 🎯 Objectives
- Improve input power factor
- Reduce Total Harmonic Distortion (THD)
- Achieve efficient AC-DC conversion
- Analyse performance using simulation

---

## 🛠️ Tools & Technologies
- MATLAB Simulink
- Power Electronics Concepts
- Control Systems
